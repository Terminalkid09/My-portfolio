# Private diary with HTML/CSS/JS

## Features
- Secure diary with password protection (set password at first access)
- Local storage of password and notes
- Gallery 
- Add, edit, delete entries note
- Quick idea and reaction heart
- Secret gallery for private images
- Logout function

## How to run
- Open 'index' in your web browser
