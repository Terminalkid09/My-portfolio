# Console password manager in C++

## feature
- Add password
- List of password
- Search password
- Generate password
- Store passwords in a local file (passwords.dat)
- Simple XOR encryption for stored passwords

### How to run
Compile: 'g++ gestore-password.cpp -o gestorePassword'
Execute: './gestorePassword o gestorePassword.exe'
