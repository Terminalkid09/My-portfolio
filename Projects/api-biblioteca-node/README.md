# API biblioteca coon Node.js

API REST per gestire i libri.

## Rotte
- GET '/api/libri' → lista libri
- POST '/api/libri' → aggiungi libro
- Put '/api/libri' → modifica libro
- DELETE '/api/libri' → elimina libro
